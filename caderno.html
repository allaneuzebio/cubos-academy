Aula de Variáveis

console.log("Mensagem") - usado para imprimir uma mensagem na tela variáveis > são como caixas que servem para guardar informações, possuem uma etiqueta. const/constante > comando para criar ou iniciar a variável, o que vier depois de const é o nome da variável, ou seja, a etiqueta da caixa. Só pode fazer 1 única atribuição à variável dentro do const. const nome = 'Allan'; -> uma variável com a etiqueta Nome e o valor Allan. Para guardar números não se usa as ASPAS, ou ele imprimirá o número na tela console.log(nome); imprimir na tela o valor da variável com a etiqueta Nome, importante notar que imprimir uma variável é SEM ASPAS. let -> comando para criar ou iniciar a variável, o que vier depois de const é o nome da variável, ou seja, a etiqueta da caixa. Permite várias atribuições, mas o que for atribuído na linha de baixo substituí o conteúdo anterior, da linha de cima.

Aula de Operadores Aritméticos

let resultado = 5 + 4; --> essa variável vai resolver a operação do lado direito (a soma) e depois vai guardar o Resultado. Também é possível fazer com letras. +,-,/,* >> ** --> dois asteriscos é usado para fazer potência, por exemplo 5**4 = cinco elevado a quarta (5x5x5x5) a sequencia de operação em contas grandes é a seguinte: primeiro *, /, +, -, ao colocar alguma operação entre parenteses () o sistema vai fazer primeiro as operações entre parenteses e depois continua na ordem original

console.log(resultado); vai imprimir na tela o conteúdo que está guardado como resultado, consequencia da operação efetuada. Exemplo de operações com letras: const x = 5; const y = 6; let resultado = 3 + x - y; resultado = resultado + 2; --> usa o resultado anterior para realizar uma nova operação, como uma calculadora, não usa outra variável, também pode ser escrito assim: 'resultado +=2' console.log(resultado);

const resto = 13 % 5; ele vai guardar o 'resto'da divisão, ele vai fazer a conta e o resultado que sobra dessa divisão vai ser guardado no valor "Resto", é chamado de Módulo.
