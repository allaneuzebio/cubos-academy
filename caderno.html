Aula de Variáveis

Tipos de Variáveis: number = números string = do tipo texto boolean = verdadeiro ou falso

console.log("Mensagem") - usado para imprimir uma mensagem na tela variáveis > são como caixas que servem para guardar informações, possuem uma etiqueta. const/constante > comando para criar ou iniciar a variável, o que vier depois de const é o nome da variável, ou seja, a etiqueta da caixa. Só pode fazer 1 única atribuição à variável dentro do const. const nome = 'Allan'; -> uma variável com a etiqueta Nome e o valor Allan. Para guardar números não se usa as ASPAS, ou ele imprimirá o número na tela console.log(nome); imprimir na tela o valor da variável com a etiqueta Nome, importante notar que imprimir uma variável é SEM ASPAS. let -> comando para criar ou iniciar a variável, o que vier depois de const é o nome da variável, ou seja, a etiqueta da caixa. Permite várias atribuições, mas o que for atribuído na linha de baixo substituí o conteúdo anterior, da linha de cima.

Aula de Operadores Aritméticos

let resultado = 5 + 4; --> essa variável vai resolver a operação do lado direito (a soma) e depois vai guardar o Resultado. Também é possível fazer com letras. +,-,/,* >> ** --> dois asteriscos é usado para fazer potência, por exemplo 5**4 = cinco elevado a quarta (5x5x5x5) a sequencia de operação em contas grandes é a seguinte: primeiro *, /, +, -, ao colocar alguma operação entre parenteses () o sistema vai fazer primeiro as operações entre parenteses e depois continua na ordem original

console.log(resultado); vai imprimir na tela o conteúdo que está guardado como resultado, consequencia da operação efetuada. Exemplo de operações com letras: const x = 5; const y = 6; let resultado = 3 + x - y; resultado = resultado + 2; --> usa o resultado anterior para realizar uma nova operação, como uma calculadora, não usa outra variável, também pode ser escrito assim: 'resultado +=2' console.log(resultado);

const resto = 13 % 5; ele vai guardar o 'resto'da divisão, ele vai fazer a conta e o resultado que sobra dessa divisão vai ser guardado no valor "Resto", é chamado de Módulo.

Operadores Condicionais (Comparação)

São operadores de comparação que são utilizados para fazer Condicionais const resultado = 5 > 4; console.log(resultado); let x = 6; let y = 5; const resultado= x > y; console.log(resultado); == --> faz uma pergunta, por exemplo se um númerp é igual ao outro ! -> negação, pergunta se são diferentes === --> pergunta se os elementos são Idênticos, mesmo tipo e valor; !== --> pergunta se os elementos são diferentes pode ser usado também com false e true nos valores

Condicionais (if (se)/else (senão))

const idade = 30; if (idade >=18) { console.log("Pode seguir, pois é maior de idade"); } (vai acontecer se for uma afirmação verdadeira, ou seja, se a mensagem for igual ou maior do que 18) else { console.log("Barrado") (vai acontecer se o resultado da variável anterior for false, ou seja, se a idade for menor do que 18)}

Condicionais Encadeadas

// Menores de 18 -> Menor de Idade // Entre 18 e 60 -> Adulto // Maior de 60 anos -> Idoso const idade = 61; if (idade < 18) { console.log("Menor de Idade"); } else if (idade <= 60) { console.log("Adulto"); } else { console.log("Idoso"); }

Primeiro ele vai verificar a primeira condição, se a idade é menor do que 18. Se o resultado for falso, ele vai verificar a segunda condição, não precisa perguntar se ele for maior que 18 e menor do que 60, por que a operação anterior já afirmou que ele é maior ou menor do que 18 anos.

Outro Exemplo

// Menores de 18 -> Menor de Idade (meia) // Entre 18 e 60 -> Adulto (meia com carteirinha) // Maior de 60 anos -> Idoso (meia) const idade = 35; const temCarteirinha = true; if (idade < 18) { console.log("Menor paga Meia"); } else if (idade <= 60) { if (temCarteirinha === true) { console.log("adulto paga meia");} else { console.log("Adulto Inteira"); } } else { console.log("Idoso paga Meia"); }

Operadores Lógicos

E / &&

// Menores de 18 -> Menor de Idade (meia) // Entre 18 e 60 -> Adulto (meia com carteirinha) // Maior de 60 anos -> Idoso (meia) //Inteira - Adulta e Não tem carteirinha // Meia - O resto //&& = E (and) const idade = 35; const temCarteirinha = true; const Adulto = idade >= 18 && idade <= 60; if (Adulto && temCarteirinha === false) { console.log("Inteira"); } else { console.log("MEIA"); } Usa o E para simplificar, por exemplo se a pessoa for adulta E não tiver carteirinha, pagará inteira, e qualquer outro resultado pagará MEIA temcCarteirinha === false -> pode ser substituído por > !temCarteirinha

OU / ||

|| = OU (or)

const idade = 35; const temCarteirinha = true; const Adulto = idade >= 18 && idade <= 60; if (idade < 18 || idade > 60 || Adulto && temCarteirinha) { console.log("MEIA") } else { console.log("INTEIRA") }
